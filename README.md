
Quadratic Equation Solver

Overview

This project is a Python program that solves quadratic equations using the quadratic formula.

The program accepts values for the coefficients a, b, and c in the equation:

ax² + bx + c = 0

It then calculates the discriminant and determines whether the equation has:

- Two real solutions
- One real solution
- No real solutions

Features

- User-friendly input system
- Calculates the discriminant
- Finds real roots using the quadratic formula
- Handles different solution cases
- Demonstrates the application of mathematics using Python

Skills Demonstrated

- Python programming
- Mathematical problem-solving
- Conditional statements (if, elif, else)
- User input handling
- Working with mathematical functions

Example

Input:

a = 1

b = -5

c = 6

Output:

x1 = 3

x2 = 2
